# Activity System Rules (V2)

Use only:

* story.md
* manifest metadata
* source_reference / scripture_reference
* must_include / must_avoid / start_boundary / end_boundary
* recent activity history

Never paste full copyrighted scripture text.

Every ActivityPack must:

* feel story-specific and worth sharing
* be printable black-and-white
* target ages 6–13
* use ordinary household materials only
* take 10–30 minutes
* never embed coloring_page.png
* never show answer keys on the printable pages
* never leave nearly blank pages
* never use tiny text
* never fall back to generic worksheets for Krishna Book stories

Page count:

* normal packs: 2–4 pages
* COLORING_ONLY or QUICK_DISCUSSION: 1 page preferred, 2 max
* never more than 5 pages

Parent clarity:

* name materials
* note scissors/glue supervision
* state send mode as OPTIONAL / WEEKEND_PROJECT / PARENT_GUIDED / SEND_NOW
