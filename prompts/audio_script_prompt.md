# Audio Script Prompt

Write a bedtime narration script for children ages 6–12.

Requirements:

- 5 to 8 minutes when read aloud.
- Natural narration voice.
- Use pause markers like [pause].
- No markdown.
- Pronounce Sanskrit names clearly and simply.
- Calm devotional bedtime tone.
- No stage directions except [pause].
- Begin with a gentle Hare Krishna greeting.
- End with a soft bedtime reflection.
