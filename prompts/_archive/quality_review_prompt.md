# Quality Review Prompt

Review the generated Krishna Book bedtime package.

Check:

- Faithful to the current queue row and summary_seed
- Age appropriate (6–12)
- No unrelated pastime mixed in
- All required sections present
- Story is not generic
- Activity sheet is story-specific
- Image prompt is specific and devotional
- Parent notes include source reference
- Moral comes directly from the pastime
- WhatsApp caption is parent-friendly and practical

Fail if the package could belong to any random story instead of the current Krishna Book chapter.
