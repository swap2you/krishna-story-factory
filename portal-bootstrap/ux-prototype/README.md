# Krishna Story Portal UX Prototype

This is a standalone, local, read-only UX prototype. It does not alter or call the Krishna Story Factory backend.

## Open locally

Double-click `index.html`, or run a local static server:

```powershell
cd <prototype-folder>
python -m http.server 8088
```

Then open:

`http://localhost:8088`

## Included prototype areas

- Public home/dashboard
- Krishna Book collection
- Story cards
- Story detail
- Custom audio player
- Speed controls and playlist/bookmark concepts
- Beautiful markdown reader
- Embedded PDF with print/download
- Coloring gallery with downloads
- Source-integrity panel
- Śloka-learning concept
- Teacher pack builder
- Local Factory Studio concept

The production actions are intentionally nonfunctional. They must be connected through a secure local API gateway in a later phase.
