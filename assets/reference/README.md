# Style Reference Images (Optional, Private)

Place approved private style-reference images here:

- `line_art_reference.png`
- `poster_reference.png`

These files guide:

- layout richness
- visual hierarchy
- line quality
- cinematic lighting
- devotional mood

They must **not** override the current story's:

- characters
- setting
- events
- expressions
- title
- quotation

Reference images are optional. Do not commit PNG/JPG files from this folder.
